i → вставка перед курсором
I → вставка в начале строки
a → вставка после курсора
A → вставка в конце строки
o → новая строка снизу
O → новая строка сверху
v → визуальный режим (символьный)
V → визуальный режим (построчный)
Ctrl + v → визуальный блок (колонки)
yy → копировать строку
dd → удалить строку
p → вставить после курсора
P → вставить перед курсором
y → копировать выделение
d → удалить выделение
>> → сдвинуть вправо (таб)
<< → сдвинуть влево
== → авто-форматнуть строку
:m+1 → переместить строку вниз
:m-2 → переместить строку вверх
/текст → поиск вперёд
?текст → поиск назад
n → следующее совпадение
N → предыдущее совпадение
:%s/old/new/g → заменить во всём файле
gg → начало файла
G → конец файла
Ctrl + d → полстраницы вниз
Ctrl + u → полстраницы вверх
